Mini Games

THE MEMORY GAME

This memory game challenges you to match pairs of Pokémon hidden on cards.

All cards are laid face down, and on each turn, you flip two cards to reveal the Pokémon. If the cards match, they remain face up; if not, they are turned back over.

The goal is to remember the locations of the Pokémon and match all pairs with as few attempts as possible.

Test your memory and see how quickly you can find every pair!

THE MASTERMIND

Mastermind is a code-breaking game where the goal is to guess a secret code.

The code consists of a sequence of colors or symbols arranged in a specific order.

After each guess, you receive feedback indicating how many colors or symbols are correct and in the correct position, and how many are correct but in the wrong position.

Use this feedback to refine your guesses and crack the code before you run out of attempts.

THE HANGMAN

The goal is simple: guess the hidden word!

To do this, you'll need to figure out which letters are in the word.

Each wrong guess adds a part to the stick figure—head, body, arms, and legs. But with every correct guess, you reveal the letters in the word.

The challenge? Guess the word before the stick figure is fully drawn, and you win!
